Regras do Jogo da Memória
O objetivo do jogo é combinar pares de cartas, onde cada par consiste em uma imagem e seu conceito correspondente.
Objetivo: O jogador deve encontrar 8 pares de cartas corretamente combinados (imagem e conceito) antes de cometer 3 erros.
Botões:
Verificação: Após virar duas cartas, o jogador pode usar o botão de verificação para confirmar se as cartas formam um par correto.
Reverter: O jogador pode usar o botão de reverter para desfazer a última jogada e virar as cartas novamente.
Pontuação:
Cada par correto de cartas vale 1 ponto.
O jogador pode atingir no máximo 8 pontos (8 pares encontrados corretamente).
O jogador tem 3 tentativas erradas. Após 3 erros, o jogo termina.
Fim de Jogo:
Vitória: Encontrar todos os 8 pares corretamente.
Derrota: Cometer 3 erros.
